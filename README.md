#  Weather App (Flask)

A simple and beautiful Weather Application built using **Flask (Python)** that fetches real-time weather data using the OpenWeather API.

---

##  Features
- Search weather by city name
- Real-time temperature (°C)
- Weather description (Clear, Rain, Clouds, etc.)
- Humidity display
- Dynamic background changes based on weather
- Responsive UI design

---

##  Tech Stack
- Python (Flask)
- HTML
- CSS
- OpenWeather API
- Requests Library
